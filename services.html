Services

  • Website Design

    Website design (desain situs web) adalah proses merancang dan membuat tampilan visual serta pengalaman pengguna dari sebuah situs web. Desain situs web melibatkan pengaturan elemen-elemen seperti tata letak, warna, tipografi, grafis, gambar, dan interaksi untuk menciptakan antarmuka yang menarik, intuitif, dan efektif.

    pricing: Rp1000 - Rp200.000

  • Website Maintanance

    Website maintenance adalah serangkaian tindakan yang dilakukan secara teratur untuk memastikan bahwa sebuah situs web tetap berfungsi dengan baik, terkini, aman, dan optimal. Ini melibatkan pemantauan, perawatan, dan peningkatan yang dilakukan pada elemen-elemen teknis dan konten situs web.

    pricing: Rp2.500.000 per month

  • Website Hosting

    Website hosting adalah layanan yang disediakan oleh perusahaan hosting web yang memungkinkan individu atau organisasi untuk membuat situs web mereka dapat diakses melalui internet. Dalam layanan ini, penyedia hosting web menyimpan file dan data situs web pada server yang terhubung ke internet sehingga pengguna dapat mengaksesnya kapan pun mereka mau.

    pricing: Rp25.000.000 per month
